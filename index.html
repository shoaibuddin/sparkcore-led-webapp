Spark Core control using jQuery / HTML!

Basic webpage to control Spark Core Dx pins using jQuery

  • Upload spark-src/firmware.ino to your Core.
  • Open index.html.
  • Enter your Core's Device ID and Access Token
  • Choose a pin number and select ON or OFF.
