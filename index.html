Spark Core control using jQuery / HTML!

Basic webapp to control Spark Core pins using jQuery/HTML

  • Upload spark-src/led-webapp.ino to your @ Core.
  • Fire index.html in browser
  • Enter your Core's Device ID and Access Token
    • Use Sparl CLI to communicate with your core
    • Use "$ spark list" to get devie ID
    • Use "$ spark login" to get access token
    • Use "$ spark login" to get access token
  • Choose a pin number and select Turn ON LED or Turn OFF LED.
